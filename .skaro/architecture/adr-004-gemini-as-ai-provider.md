# ADR-004: Использование Gemini API как единственного AI-провайдера для перевода, TTS, Summary и X-Ray

**Status:** accepted
**Date:** 2026-03-29

## Context
Приложение позиционируется как «AI-first» ридер. Необходимы: перевод текста, синтез речи, создание саммари, X-Ray (анализ персонажей/мест), словарь. Все функции должны работать в рамках единого API с управляемой квотой и ключом, хранящимся в Keychain.

## Decision
Gemini API выбран как единственный AI-провайдер для всех интеллектуальных функций. API ключ хранится исключительно в `KeychainManager.shared`. Все запросы проходят через `GeminiService` → `AIRequestQueue` → `RateLimiter`. Квота отслеживается через `QuotaTracker`. Таймауты: интерактивные запросы 10s, фоновые 30s. При отсутствии сети — немедленный `.offline` error с понятным UI-баннером. ElevenLabs исключён из текущего milestone (возможен в milestone 09 как ultra-premium голос).

## Alternatives
1. **OpenAI GPT-4 + Whisper:** Зрелая экосистема с широкой поддержкой — отклонён, так как Gemini предоставляет мультимодальные возможности (текст + аудио) в едином API, что упрощает архитектуру. Gemini имеет более выгодное ценообразование для мобильных приложений.
2. **Несколько AI-провайдеров (OpenAI + ElevenLabs + Anthropic):** Резервирование и специализация — отклонён, так как значительно усложняет архитектуру, управление ключами, квотами и circuit breaker логикой. Увеличивает поверхность атаки для утечки credentials.
3. **On-device ML (Core ML / Apple Intelligence):** Полная оффлайн-работа, приватность — отклонён для текущего milestone, так как качество on-device моделей для перевода и саммари значительно уступает облачным, а размер моделей критичен для мобильного приложения.

## Consequences
- Positive: Единый API упрощает управление ключами, квотами и мониторингом. Мультимодальность Gemini позволяет реализовать TTS и перевод через один сервис. Снижается сложность `AIRequestQueue`.
- Negative: Жёсткая зависимость от одного провайдера (vendor lock-in). Все AI-функции недоступны оффлайн. При деградации Gemini API все AI-функции одновременно становятся недоступными.
- Risks: Изменение ценовой политики Google может сделать сервис нерентабельным. Региональные ограничения доступа к Gemini API могут затронуть пользователей в отдельных странах. Circuit breaker (3 ошибки → `.degraded`) снижает риск каскадных сбоев.
