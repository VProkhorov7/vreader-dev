# Ядро ридера: чтение текста, PDF и комиксов

Реализация основного ридера с поддержкой EPUB, FB2, TXT, PDF и CBZ/CBR форматов, панелей управления с frosted glass, аннотациями и настройками отображения. После этого этапа пользователь может читать книги основных форматов.
