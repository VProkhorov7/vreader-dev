# Specification: text-reader-view

## Context
TextReaderView — основной компонент для чтения текстовых форматов. Поддерживает горизонтальную пагинацию (листание), настройки шрифта/размера/интервала из ReaderState.displaySettings, и автоматическое сохранение позиции.

## User Scenarios
1. **Пользователь читает EPUB:** Листает страницы горизонтальным свайпом, позиция сохраняется автоматически.
2. **Пользователь меняет размер шрифта:** Текст перерасчитывается, позиция сохраняется.
3. **Пользователь делает long press на слове:** Появляется меню: Перевести, Добавить в словарь, Выделить.

## Functional Requirements
- FR-01: TabView с PageTabViewStyle для горизонтальной пагинации
- FR-02: Каждая страница — отдельный View с текстом из FileFormatHandler.openPage()
- FR-03: Применение ReaderState.displaySettings: fontSize, lineSpacing, fontFamily из темы
- FR-04: Автосохранение позиции: при смене страницы → ReaderState.updatePosition()
- FR-05: Жест двойного тапа: показать/скрыть панели управления
- FR-06: Long press на тексте: UITextView selection menu с опциями Translate, Highlight, Note
- FR-07: Предзагрузка следующей страницы в фоне
- FR-08: Поддержка RTL текста (для арабских книг в будущем)
- FR-09: Отображение номера страницы и прогресса
- FR-10: Поддержка Dynamic Type
- FR-11: Все строки через L10n.Reader.*

## Non-Functional Requirements
- NFR-01: Листание страниц 60fps без jank
- NFR-02: Предзагрузка следующей страницы < 500ms

## Boundaries (что НЕ входит)
- Не реализовывать TTS на этом этапе
- Не реализовывать синхронизацию позиции через CloudKit
- Не реализовывать поиск по тексту

## Acceptance Criteria
- [ ] Горизонтальная пагинация работает
- [ ] Настройки шрифта применяются
- [ ] Позиция сохраняется при смене страницы
- [ ] Long press меню появляется
- [ ] Предзагрузка следующей страницы работает
- [ ] Dynamic Type поддерживается
