# Specification: library-view

## Context
Главный экран приложения. Сетка книг с поиском, фильтрами, сортировкой. Производительность: P95 < 300ms для 1000 книг. Пагинация при > 500 книг.

## User Scenarios
1. **Пользователь открывает библиотеку:** Видит сетку книг, отсортированных по дате добавления.
2. **Пользователь ищет 'Толстой':** Фильтрует по автору в реальном времени.
3. **Пользователь фильтрует по жанру Fantasy:** Отображаются только книги жанра Fantasy.
4. **Библиотека содержит 1000 книг:** Загрузка < 300ms, пагинация работает плавно.

## Functional Requirements
- FR-01: `LibraryView` — SwiftUI View с `@Query` для загрузки книг.
- FR-02: Три вкладки: Library (все книги), Favorites (isFinished или помечены), Collections (коллекции).
- FR-03: Сетка: `LazyVGrid` с адаптивными колонками (2 на iPhone, 4 на iPad).
- FR-04: Поиск: `searchable` modifier, фильтрует по title, author, tags в реальном времени.
- FR-05: Фильтры: жанр, формат, contentState, статус прочтения.
- FR-06: Сортировка: по дате добавления, по названию, по автору, по прогрессу.
- FR-07: Пагинация при > 500 книг через `@Query` с `fetchLimit` и `fetchOffset`.
- FR-08: Pull-to-refresh для синхронизации с облаком.
- FR-09: FAB (Floating Action Button) для импорта книги через `fileImporter`.
- FR-10: Пустое состояние: иллюстрация + кнопка импорта если библиотека пуста.
- FR-11: Все строки через `L10n.*`.
- FR-12: Тема через `@Environment(\.appTheme)`.
- FR-13: `fileImporter` поддерживает все форматы из `BookFormat` через UTType (optional binding).

## Non-Functional Requirements
- NFR-01: Загрузка 1000 книг P95 < 300ms.
- NFR-02: Scroll 60fps без jank.

## Boundaries (что НЕ входит)
- Не реализовывать CollectionsView детально — только список коллекций.
- Не реализовывать MetadataEditorView.

## Acceptance Criteria
- [ ] Сетка книг отображается корректно на iPhone и iPad.
- [ ] Поиск работает в реальном времени.
- [ ] Фильтры применяются корректно.
- [ ] Пагинация работает при > 500 книг.
- [ ] Импорт через FAB запускает BookImporter.
- [ ] Производительность P95 < 300ms для 1000 книг.
