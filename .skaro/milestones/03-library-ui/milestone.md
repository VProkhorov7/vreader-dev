# UI библиотеки: карточки, список, навигация

Реализация полного UI библиотеки: LibraryView с сеткой книг, BookCardView с badge состояний, поиском, фильтрами и детальным просмотром книги. После этого этапа пользователь может видеть свою библиотеку и управлять книгами.
