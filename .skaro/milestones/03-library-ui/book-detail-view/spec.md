# Specification: book-detail-view

## Context
BookDetailView показывается при тапе на карточку книги. Содержит полные метаданные, большую обложку, кнопки действий в зависимости от contentState, прогресс чтения и список аннотаций.

## User Scenarios
1. **Пользователь тапает на книгу:** Открывается детальный вид с обложкой, описанием, кнопкой "Читать".
2. **Книга в состоянии cloudOnly:** Кнопка "Скачать" + прогресс загрузки.
3. **Книга прочитана на 45%:** Показывается прогресс бар и кнопка "Продолжить чтение".

## Functional Requirements
- FR-01: Большая обложка (верхняя треть экрана) с blur background
- FR-02: Метаданные: title, author, genre, series, format, fileSize, addedAt, lastOpenedAt
- FR-03: Описание книги с expand/collapse если > 3 строк
- FR-04: Кнопка действия зависит от contentState: cloudOnly → "Скачать", previewed → "Читать превью" + "Скачать полностью", downloaded → "Читать" (или "Продолжить" если progress > 0)
- FR-05: Прогресс чтения: процент + страница
- FR-06: Список аннотаций (закладки, хайлайты) с переходом к позиции
- FR-07: Кнопка "Редактировать метаданные" → MetadataEditorView (sheet)
- FR-08: Кнопка "Удалить" с подтверждением
- FR-09: Теги отображаются как chips
- FR-10: Все строки через L10n
- FR-11: Поддержка VoiceOver

## Non-Functional Requirements
- NFR-01: Открытие детального вида < 200ms
- NFR-02: Плавная анимация появления

## Boundaries (что НЕ входит)
- Не реализовывать MetadataEditorView (только заглушка-sheet)
- Не реализовывать ридер
- Не реализовывать аннотации (только отображение существующих)

## Acceptance Criteria
- [ ] Детальный вид открывается при тапе на карточку
- [ ] Кнопки действий соответствуют contentState
- [ ] Прогресс чтения отображается
- [ ] Удаление с подтверждением работает
- [ ] Все строки через L10n
- [ ] VoiceOver поддерживается
