# Аудиоплеер: MP3/M4B и фоновое воспроизведение

Реализация AudioPlayerView для аудиокниг с поддержкой фонового воспроизведения, Control Center, Now Playing Info и базового TTS через AVSpeechSynthesizer. После этого этапа пользователь может слушать аудиокниги.
