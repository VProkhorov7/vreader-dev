# Specification: quote-export

## Context
Пользователи хотят делиться цитатами из книг в социальных сетях. Экспорт: красивое изображение с цитатой и обложкой книги, текст, Markdown.

## User Scenarios
1. **Пользователь выделяет цитату:** Нажимает 'Поделиться' → выбирает формат → share sheet.
2. **Пользователь экспортирует все аннотации книги:** Получает Markdown файл со всеми заметками.
3. **Пользователь создаёт красивую карточку цитаты:** Изображение с обложкой, цитатой, автором.

## Functional Requirements
- FR-01: `QuoteExporter` — final class.
- FR-02: `exportAsImage(annotation:book:) async -> UIImage` — красивая карточка с обложкой.
- FR-03: `exportAsText(annotation:book:) -> String` — форматированный текст.
- FR-04: `exportAllAnnotations(book:format:) async -> Data` — все аннотации в Markdown или TXT.
- FR-05: Карточка: обложка книги (blur background), цитата, автор, название книги.
- FR-06: Применяет текущую тему к карточке.
- FR-07: `ShareLink` для экспорта через системный share sheet.
- FR-08: Кнопка 'Поделиться' в NotesPanel и контекстном меню выделения.

## Non-Functional Requirements
- NFR-01: Генерация изображения < 1s.

## Boundaries (что НЕ входит)
- Не реализовывать прямую публикацию в социальные сети.
- Не реализовывать синхронизацию аннотаций с Readwise.

## Acceptance Criteria
- [ ] Карточка цитаты генерируется корректно.
- [ ] Экспорт всех аннотаций в Markdown работает.
- [ ] Share sheet открывается с правильным контентом.
