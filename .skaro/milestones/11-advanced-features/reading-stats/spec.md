# Specification: reading-stats

## Context
Статистика чтения на основе `ReadingStatsRecord`. Отображает: время чтения за день/неделю/месяц, страницы, слова, прогресс по книгам. Мотивирует пользователя читать больше.

## User Scenarios
1. **Пользователь открывает статистику:** Видит графики чтения за последние 7 дней.
2. **Пользователь завершает книгу:** Статистика обновляется, показывает достижение.
3. **Пользователь видит streak:** Количество дней подряд с чтением.

## Functional Requirements
- FR-01: `ReadingStatsView` — SwiftUI View в SettingsView или отдельной вкладке.
- FR-02: График чтения за 7/30/365 дней (Chart framework, iOS 16+).
- FR-03: Метрики: минуты чтения, страницы прочитано, слов прочитано, книг завершено.
- FR-04: Streak: количество дней подряд с чтением (минимум 10 минут).
- FR-05: Топ-5 жанров по времени чтения.
- FR-06: Данные из `ReadingStatsRecord` через `@Query`.
- FR-07: `ReadingStatsService` — вычисляет агрегированные метрики из `ReadingStatsRecord`.
- FR-08: Все строки через `L10n.*`.

## Non-Functional Requirements
- NFR-01: Загрузка статистики < 500ms.

## Boundaries (что НЕ входит)
- Не реализовывать экспорт статистики.
- Не реализовывать gamification (badges, achievements).

## Acceptance Criteria
- [ ] Графики отображаются корректно.
- [ ] Streak вычисляется правильно.
- [ ] Данные актуальны после сессии чтения.
