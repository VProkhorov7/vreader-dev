# Specification: auto-scroll

## Context
Автоскролл позволяет читать без касания экрана. Скорость настраивается. Пауза при тапе или скролле пользователя. Premium функция.

## User Scenarios
1. **Пользователь включает автоскролл:** Текст плавно прокручивается с выбранной скоростью.
2. **Пользователь тапает на экран:** Автоскролл паузируется, controls появляются.
3. **Пользователь регулирует скорость:** Слайдер в ReaderSettingsPanel.

## Functional Requirements
- FR-01: `AutoScrollController` — @Observable class, управляет автоскроллом.
- FR-02: `start(speed:)` / `pause()` / `stop()` методы.
- FR-03: Скорость: 1-10 (условные единицы), настраивается в ReaderSettingsPanel.
- FR-04: Пауза при: тапе, свайпе, появлении keyboard.
- FR-05: Возобновление через 3 секунды после последнего взаимодействия (настройка).
- FR-06: Premium only — проверка через `PremiumGate`.
- FR-07: Плавная анимация скролла (не прыжками).
- FR-08: Кнопка автоскролла в ReaderBottomBar.

## Non-Functional Requirements
- NFR-01: Скролл 60fps без jank.

## Boundaries (что НЕ входит)
- Не реализовывать автоскролл для ComicReaderView.
- Не реализовывать eye-tracking автоскролл.

## Acceptance Criteria
- [ ] Автоскролл работает плавно.
- [ ] Пауза при взаимодействии работает.
- [ ] Скорость настраивается.
- [ ] Premium gate соблюдается.
