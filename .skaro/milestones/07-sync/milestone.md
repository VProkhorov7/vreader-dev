# Синхронизация: CloudKit и оффлайн очередь

Реализация синхронизации аннотаций через CloudKit, позиции чтения через NSUbiquitousKeyValueStore, conflict resolution с lamportClock и фоновой синхронизации. После этого этапа данные синхронизируются между устройствами пользователя.
