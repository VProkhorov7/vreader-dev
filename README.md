# VReader

Cross-platform application for structured content consumption and clean reading experience.

---

## Overview

VReader is a reading application focused on clarity, simplicity, and structured content delivery across platforms.

---

## Features

- Clean reading interface
- Structured content display
- iCloud synchronization (Apple platforms)
- Lightweight architecture

---

## Platforms

- iOS (current)
- macOS (current)
- Android (planned)

---

## Tech stack

- Swift (Apple platforms)
- iOS / macOS (Xcode)
- iCloud integration

---

## Project structure

```
Vreader/
VReader.xcodeproj
```

---

## Notes

- Core logic is located inside `Vreader/`
- Designed for scalability and future cross-platform expansion

---

## Vision

To create a focused, minimal, cross-platform reading environment with strong structure and extensibility.
